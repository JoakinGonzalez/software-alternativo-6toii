Examen Recuperatorio Software Alternativo

1. a. Un servidor, como la misma palabra indica, es una computadora informática que está al servicio de otras computadoras, computadores o personas llamadas clientes y que le suministran a estos b. Los servidores FTP son las aplicaciones de software que permiten la transferencia de archivos de un dispositivo a otro. Realizan dos simples tareas obtener y poner. En resumen, puedes obtener archivos del servidor FTP o poner archivos en dicho servidor c. Dirección IP: La dirección IP es la identificación de un dispositivo en una red; estos se comunican entre sí gracias a la misma. Máscara de subred: Una máscara de red es una combinación de bits que permite delimitar el ámbito de una red de computadoras Puerta de enlace: Una puerta de enlace es un equipo o elemento que perteneciendo a la red está configurado para dotar a los PC que integran dicha red puedan salir al exterior, generalmente realizando para ello operaciones de traducción de direcciones IP d.
